# 🐙 가평 여행 게임방

반 친구 8명이 폰으로 함께 즐기는 **신서유기 스타일 실시간 게임 진행 도우미**.
사이트에 들어가 자기 이름을 고르면 입장하고, 진행자가 게임을 고르면 각자 폰에
개인 미션/멘트가 실시간으로 뜹니다.

## 기능

- **이름 선택 입장** — 8명 명단([players.js](players.js))에서 탭 한 번으로 입장 (방 코드 없음)
- **실시간 동기화** — Socket.IO로 모든 폰이 같은 상태 공유
- **개인 미션 분배** — 게임마다 각 폰에 다른 역할/멘트/비밀단어 전송 (예: 고요 속의 외침에서 출제자만 단어를 봄)
- **진행자 컨트롤** — 게임 선택, 점수 ±, 정답 확인(진행자만), 점수 초기화
- 한 폰에서 플레이어 + 진행자 동시 사용 가능

## 로컬 실행

```bash
npm install
npm start
```

- 이 컴퓨터: http://localhost:3000
- 같은 와이파이/핫스팟의 폰: `http://<컴퓨터IP>:3000`

## 배포 (Render 무료)

1. 이 레포를 GitHub에 푸시
2. [Render](https://render.com)에서 GitHub로 로그인 → New → Blueprint → 이 레포 선택
3. [render.yaml](render.yaml)을 자동 인식해서 배포 → `https://<이름>.onrender.com` 발급
4. 그 링크를 친구들 폰에서 열면 끝

> 무료 플랜은 15분 미사용 시 잠들어, 첫 접속에 30초~1분 깨우는 시간이 걸립니다.

## 게임 추가

[games.js](games.js)에 항목 하나 추가하면 엔진이 자동으로 미션 분배·공유 화면·정답 표시를 처리합니다.
